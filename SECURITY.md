# Security Policy

## Scope

This is a static frontend application served via GitHub Pages. There is no backend, database, or user authentication. The Python pipeline runs locally and does not expose network services.

## Reporting a Vulnerability

If you find a security issue (e.g., XSS in the player, unsafe file handling in the pipeline), please report it by opening a GitHub issue or emailing the maintainer directly.

Since this project has no server-side components or user data, most web vulnerabilities do not apply. However, we take reports seriously and will respond promptly.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest (master) | Yes |
| Older commits | No |
